# Powerplant_ANN_Regression
Implementation of ANN(Artificial Neural Network) using PyTorch to predict energy output of power plant

## DATASET
Dataset is given and it has 9568 samples , 4 features

## Model Architecture
-> Input Layer : 4 neurons
-> Hidden Layers : 2 layers with ReLU as activation function
-> Output Layer : 1 neuron

## Results
-> R2_Score : 0.9317
-> Optimizer : Adam
->Loss : MSE

## Libraries used
PyTorch,Pandas,Scikit-learn,Matplotlib
